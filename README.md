# Employee Management System

## 📌 Project Description

This is a web-based Employee Management System built using Django.  
It allows managing employees, jobs, news, and other HR-related activities.

---

## 🚀 Features

- Add, update, delete employees
- View employee details
- Job openings management
- Latest news section
- Calendar & holidays
- User login system

---

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS

---

## 📂 Project Structure

- emp/ → main app (models, views, forms)
- templates/ → HTML files
- static/ → CSS & images
- manage.py → project runner

---

## ▶️ How to Run Project

1. Clone repository:
   git clone https://github.com/Testing-it-github/empsystem.git

2. Go to folder:
   cd empsystem

3. Install requirements:
   pip install -r requirements.txt

4. Run server:
   python manage.py runserver

5. Open in browser:
   http://127.0.0.1:8000/
